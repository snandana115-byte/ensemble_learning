📄 Project Description

📊 This project applies Ensemble Learning techniques to predict whether a TV show is highly rated using a dataset of top-rated TV shows. The dataset includes features such as vote count, popularity, and other key attributes that influence audience reception.

🎯 The main objective is to build a classification model that identifies whether a show belongs to the “highly rated” category. For this purpose, the Random Forest algorithm (a Bagging-based ensemble method) is implemented to improve accuracy and reduce overfitting.

🧹 The data is carefully preprocessed by handling missing values, selecting important features, and splitting it into training and testing sets. The model is then trained and evaluated using performance metrics such as accuracy score.

⚖️ To understand the effectiveness of ensemble learning, the Random Forest model is compared with other algorithms like Decision Tree and Logistic Regression. This comparison shows that ensemble methods provide better stability and higher performance.

🚀 Finally, the trained model is saved and deployed using Streamlit, where users can input values like vote count and popularity to predict whether a TV show is highly rated.
